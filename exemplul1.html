Exemplul 1

Generare tabel folosind tagurile pentru cap de tabel si populand cu elemente preluate din Javascript!

Cod HTML

<!DOCTYPE HTML>
<html>
    <head>
        <link rel="stylesheet" href="stiluri.css">
        <script src="initiere.js"> </script>
    </head>
<body>
   <h2>Tabel cu elemente preluate din Javascript!</h2>

   <table style="width:100%">
       <tr>
          <th>Numele tau</th>
          <th>Alt nume</th>
          <th>Sau poate</th>
       </tr>
       <tr>
          <td id="numejs"></td>
          <td>Sergiu</td>
          <td>Pavel</td>
       </tr>
   </table>

<p><TH> defineste capul de tabel!lt;/p>
<p>Pentru a intelege mai bine exemplul, am adaugat margini pentru tabelul creat.lt;/p>
</body>
</html>

Cod CSS

table, th, td {
    border:1px solid black;
}
