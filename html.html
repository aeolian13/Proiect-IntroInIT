Hyper Text Markup Language (abv. HTML)

  • HTML5 este cea mai nouă versiune HTML
  • este compus din tag-uri (sau etichete, în limba română) care descriu părți diferite ale unei pagini
  • tag-urile HTML sunt plasate între simbolurile “mai mic” și “mai mare”: <eticheta>….</eticheta>
  • majoritatea tag-urilor sunt perechi. Fiecare pereche are o etichetă de deschidere și una de închidere:
    • <tag> – acesta este tag-ul de deschidere;
    </tag> – acesta este tag-ul de închidere.
  • Exemple de etichete:
    • <p> … </p> - crearea de paragrafe
    <h1> … </h1> - crearea unui titlu (exista și h2 …h6 - subtitluri)
  • Orice pagină HTML trebuie să aibă un titlu de nivel 1, adica <h1>.
  • De reținut că trebuie să aibă numai un H1, pentru SEO – adica optimizarea pentru motoarele de căutare.
  • Poate în schimb să aibă oricâte subtitluri <h2>…<h6>.
